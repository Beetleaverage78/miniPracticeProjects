Proximity

Proximity helps create organisation.

By grouping similar elements together or in close proximity, you create a relationship between those elements.

It also provides a focal point and can give the reader an idea of where they should start and finish reading.

Proximity doesn’t mean that elements have to be placed together, it means they should be visually connected in some way.

This can be by use of point size, font, colour etc… Using CSS to have an image placed amongst the text with a text wrap or image float left ) uses Proximity to create a design feature.
